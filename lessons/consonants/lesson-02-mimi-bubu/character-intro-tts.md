# 2레슨 미미와 부부: 미음 비읍 입술로 느끼는 자음 캐릭터 자기소개 TTS 대본

## TTS 방향

- 목소리: 밝고 다정한 유아 교사 톤
- 속도: 천천히, 한 문장 뒤 1초 쉬기
- 글자: 낱자 기호 대신 기역, 니은처럼 자음 이름으로 읽기
- 반복 소리: 한국어 음절 반복 대신 [g] [g] [g]처럼 발음기호로 읽기
- 호흡: 아이가 따라 말할 수 있도록 질문 뒤 짧게 멈추기
